# vim-commands
A list of commonly used VIM commands.

## General Commands

| Command | Description |
| ------- | ----------- |
| `i` | Insert Mode |
| `Esc` | Command Mode |
| `/<search_term>` | Search and then cycle through matches with n and N |

## Spell Check

| Command | Description |
| ------- | ----------- |
| `]s` | Next misspelled word |
| `[s` | Previous misspelled word |
| `z=` | Get alternatives |
| `zg` | Add word to dictionary |
